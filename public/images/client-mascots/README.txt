CLIENT MASCOT IMAGES FOLDER
============================

This folder contains images for the actual mascots deployed by each client.

NAMING CONVENTION:
------------------
Use the mascot ID from the bots.json file with lowercase format:

JUMBO MASCOTS:
- m1-liza.png (Customer Service - Liza)
- m2-remco.png (Customer Service - Remco)
- m3-sarah.png (HR - Sarah)
- m4-max.png (Store - Max)

HITAPES MASCOTS:
- m5-vinny.png (Main - Vinny)
- m6-tech.png (Main - Tech)

HAPPINESSBUREAU MASCOTS:
- m7-stella.png (Sales - Stella)
- m8-joy.png (Personal - Joy)

ALTERNATIVE NAMING (if you prefer client-specific):
----------------------------------------------------
You could also organize by client:
- jumbo-liza.png
- jumbo-remco.png
- jumbo-sarah.png
- jumbo-max.png
- hitapes-vinny.png
- hitapes-tech.png
- happiness-stella.png
- happiness-joy.png

RECOMMENDED: Use the ID-based naming (first option) as it matches the database IDs.

IMAGE SPECIFICATIONS:
---------------------
- Format: PNG (or JPG/WEBP for better compression)
- Recommended size: 400x400px or 600x600px (square aspect ratio)
- Background: Transparent or matching client brand colors:
  * Jumbo: Yellow (#FFD700)
  * HiTapes: Light Purple (#A855F7 or similar light purple)
  * Happinessbureau: Red (#EF4444)
